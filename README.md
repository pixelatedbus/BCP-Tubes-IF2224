# Better Call Pascal

Sebuah kompiler sederhana untuk bahasa pemrograman Pascal-S yang dibangun sebagai bentuk pengerjaan Tugas Besar IF2224 Teori Bahasa Formal Otomata. Program ini ditulis dengan Python.

---

## Requirements

- [Python](https://www.python.org/downloads/)

## How to Run Locally

```bash
cd src
py ./main.py
```
Masukkan input nama file .pas yang terdapat di dalam folder test/milestone-1, hasil tokenisasi akan disimpan sebagai tokens.txt di test/milestone-1, dan error akan di output di terminal.
